# Contributing to Mandiol

Thank you for wanting to help. This project exists to support women in difficult circumstances — contributions of any kind are welcome and valued.

## Who can contribute?

- Developers (frontend, mobile, backend)
- Legal professionals (solicitors, barristers, legal researchers)
- VAWG practitioners (ISVAs, SARC workers, DA advocates)
- UX/UI designers
- Translators
- Researchers and content reviewers

## What we need most

1. **Legal content review** — Laws change. If you spot outdated information, please raise an issue.
2. **Accessibility improvements** — Screen reader support, contrast, font sizing.
3. **Translations** — Priority languages: Polish, Urdu, Punjabi, Romanian, Bengali.
4. **React Native port** — Converting the HTML prototype to a proper mobile app.
5. **Safety tracker** — Real GPS integration with SMS fallback.
6. **Support agency data** — Keeping phone numbers and details current.

## Ground rules

- Be kind. Always.
- Never compromise the Quick Exit or authentication features.
- All legal content changes must be reviewed and cited.
- No advertising, tracking, or analytics SDKs.
- Follow the trauma-informed design principles in CLAUDE.md.

## How to submit

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature-name`
3. Make changes
4. Test thoroughly (especially Quick Exit and auth)
5. Submit a Pull Request with a clear description

## Reporting issues

Please use GitHub Issues. For sensitive content concerns, email the maintainer directly.
